Description:

This web-app performs two get calls to Google and Apple API's with a given search term. The app
displays 5 results from each service and sorts them alphabetically.


Technologies:

This project was developed with JDK 1.8, Spring Boot and a simple Jquery + Bootstrap front end
to provide a quick and efficient implementation.

Installation:

The entire project can be downloaded and run on command line following the guidelines found at:
https://docs.spring.io/spring-boot/docs/current/reference/html/using-boot-running-your-application.html
